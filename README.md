# Under the Sakura 🌸

A starter visual novel project for Ren'Py.

## Run
1. Install Ren'Py SDK
2. Open Ren'Py Launcher
3. Select this project folder: `under-the-sakura`
4. Launch project

## Story premise
A soft, emotional campus-night story about memory, distance, and choices beneath blooming sakura trees.
